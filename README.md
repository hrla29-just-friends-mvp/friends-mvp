# friends-mvp

App name tentative.

### Usage

```
git clone https://github.com/hrla29-just-friends-mvp/friends-mvp.git
cd friends-mvp

git checkout -b your_github_handle
yarn install
yarn run db:generate
```

To run the app in Expo:
```
yarn start
```

#### In case you need to reset database
```
yarn run db:regenerate
```
